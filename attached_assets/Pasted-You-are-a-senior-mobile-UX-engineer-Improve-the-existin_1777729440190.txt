You are a senior mobile UX engineer. Improve the existing ClientFlow app to make data input extremely fast and frictionless for freelancers using their phone.

## 🎯 Goal

Reduce the time to:

* Add a client → under 5 seconds
* Add a booking → under 5 seconds
* Send a reminder → 1 tap

---

## 🔥 UX Improvements to Implement

### 1. Quick Add Client (CRITICAL)

* Replace full form with:

  * Name (required)
  * Phone (required)
* Everything else optional or added later

Add:

* Floating Action Button (FAB) → “+”
* Opens a **minimal modal**, not full screen

---

### 2. Smart Defaults

* Auto-fill:

  * Today’s date for bookings
  * Default status = "Lead"
* Remember last used service type

---

### 3. One-Tap Actions on Client Card

Each client card should have:

* 📩 WhatsApp button
* 💰 Mark as Paid
* 📅 Add Booking

No navigation required

---

### 4. Inline Editing

* Tap any field → edit instantly
* Avoid separate “edit screen”

---

### 5. Quick Status Change (IMPORTANT)

* Replace dropdown with:

  * Swipe or tap chips:
    Lead → Quoted → Booked → Completed → Paid

---

### 6. “Add Booking” in 1 Screen

* Date (default today)
* Time
* Save

No extra fields unless expanded

---

### 7. Reduce Typing

* Use:

  * Dropdown suggestions
  * Recently used values
  * Autofill phone/name if possible

---

### 8. Dashboard = Action Center

Instead of static info:

* Show:

  * “2 payments overdue” → button → send reminders
  * “1 booking today” → quick view

---

## 🎨 UI Requirements

* Big buttons
* Thumb-friendly layout
* Minimal text
* No clutter
* Prioritize speed over aesthetics

---

## 🧠 Important Rules

* Every action ≤ 2 taps
* Avoid multi-step forms
* No unnecessary screens

---

## 🚀 Deliverables

* Updated UI components
* Refactored screens
* Code changes clearly explained
* Before/after UX explanation

Now implement these improvements in the existing codebase.
