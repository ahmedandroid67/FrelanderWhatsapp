You are a senior full-stack mobile engineer. Build a production-ready Android application called **ClientFlow** — a lightweight CRM for freelancers (starting with photographers as the primary niche).

## 🎯 Core Goal

Create a **simple, fast, offline-first app** that helps freelancers:

* Track clients
* Manage bookings
* Send WhatsApp reminders
* Track payments
* Generate simple invoices

The app must feel like a **daily assistant**, not complex software.

---

## 🛠️ Tech Stack Requirements

### Frontend (Android)

* React Native with Expo (preferred) OR Kotlin (if needed)
* Clean, minimal UI
* Optimized for low-end devices

### Backend

* Supabase:

  * PostgreSQL database
  * Auth (optional, allow guest mode first)
  * Storage (for invoices PDF later)
  * Realtime (optional for sync)

---

## 📦 Core Features (MVP)

### 1. Client Management

* Create, edit, delete client
* Fields:

  * name (required)
  * phone (required)
  * service type (e.g. wedding shoot, tutoring session)
  * notes
* Show clients in a simple list

---

### 2. Client Pipeline (IMPORTANT)

Each client has a status:

* Lead
* Quoted
* Booked
* Completed
* Paid

UI:

* Show as tags or step indicator
* Allow quick status change

---

### 3. Booking System

* Add booking to client:

  * date
  * time
  * location
  * notes

---

### 4. Payment Tracking

* Fields:

  * total amount
  * amount paid
  * due date
* Auto calculate:

  * remaining balance
* Status:

  * unpaid / partial / paid

---

### 5. Smart Reminders (CORE FEATURE)

* Show dashboard:

  * payments due today
  * overdue payments
  * upcoming bookings

---

### 6. WhatsApp Quick Actions (CRITICAL)

Add button on each client:

* “Send Reminder”
* “Send Invoice”
* “Confirm Booking”

Use:

* WhatsApp deep linking (wa.me links)

Pre-filled messages:
Example:
"Hi {name}, just a reminder your payment of {amount} is due today."

---

### 7. Simple Invoice Generator

* Generate basic invoice data:

  * client name
  * service
  * amount
  * date
* Display in-app
* (Optional) export as PDF later

---

### 8. Dashboard (HOME SCREEN)

Show:

* Today’s bookings
* Overdue payments
* Pending clients

---

## 🧱 Database Schema (Supabase)

Create tables:

### clients

* id (uuid, primary key)
* name (text)
* phone (text)
* service_type (text)
* notes (text)
* status (text)
* created_at (timestamp)

---

### bookings

* id
* client_id (foreign key)
* date
* time
* location
* notes

---

### payments

* id
* client_id (foreign key)
* total_amount
* paid_amount
* due_date
* status

---

### invoices

* id
* client_id
* amount
* description
* created_at

---
